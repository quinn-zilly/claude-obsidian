# Decision by Sampling (DbS)

Model of choice (Stewart et al. 2006, 2015): people make decisions by **sampling from memory** comparisons to prior outcomes and comparing them to current options.

## Core Mechanism

1. Attend to current options' attributes (e.g., payoff amount, probability)
2. Sample comparisons from memory — previous options encountered in similar contexts
3. Use ordinal rank of current option in sampled set to determine its subjective value

## Implications

- **Loss aversion** emerges as artifact: if negative outcomes are more salient / frequent in memory, losses rank higher → appear to have more weight. Loss aversion is not a stable parameter but a context-sensitive memory effect.
- **CPT parameters** are not fixed properties of the decision maker — they vary with the sampling distribution, which varies with task context and life experience.
- Explains why CPT parameters are so variable across studies (different sampling contexts)

## Key Finding

Broomell & Bhatia (2014) used **information theory** to show that standard experience-based stimuli (in description-experience gap studies) cannot in principle reveal probability underweighting. The stimuli don't contain enough information to estimate CPT weights. This resolves the paradox: experience-based choices don't contradict CPT because they're not testing the same thing.
