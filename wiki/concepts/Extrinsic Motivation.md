# Extrinsic Motivation

Performing an activity to attain some separable outcome. Contrasts with [[Intrinsic Motivation]] (doing for inherent satisfaction).

**Key SDT claim**: Extrinsic motivation is NOT invariantly non-autonomous. It varies greatly in its relative autonomy — from external compliance to fully integrated self-regulation.

## Self-Determination Continuum (OIT)

Per [[Organismic Integration Theory]]:

| Regulatory Style | Locus | Description |
|-----------------|-------|-------------|
| **External** | External | Satisfy demand/reward; experienced as controlled/alienated |
| **Introjected** | Somewhat external | Avoid guilt/anxiety; ego-involvement; contingent self-esteem; internally driven but not owned |
| **Identified** | Somewhat internal | Conscious valuing; personally important; accepted as own |
| **Integrated** | Internal | Fully assimilated; congruent with other values and needs; shares qualities with intrinsic |

Note: Identified + integrated are still *extrinsic* because done for separable outcomes, not inherent enjoyment. But they are autonomous extrinsic motivations.

## Internalization Process

As people internalize and integrate regulations → experience greater autonomy. Process is not strictly developmental stages (not required to progress through each level). Can internalize at any point depending on prior experience and current context.

**Conditions promoting integration** (deepest form): autonomy support + competence support + relatedness. See [[Internalization (SDT)]].

## Outcomes by Type

Ryan & Connell (1989): quasi-simplex intercorrelation pattern confirms underlying continuum.
- More external regulation → less interest/value/effort; more responsibility disowning
- Introjected → more effort but also more anxiety; poorer failure coping
- Identified → more interest/enjoyment; better coping; more effort

More autonomous (identified/integrated) extrinsic motivation → greater engagement, better performance, lower dropout, higher quality learning, better health adherence.
