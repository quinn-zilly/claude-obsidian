# Chunking

The grouping of individual elements into larger meaningful units ("chunks") that function as single items in memory (Miller 1956; Chase & Simon 1973).

## The classic chess account

Chase & Simon (1973): chess experts recall **structured** positions far better than novices but do **no better on random** positions. They argued the number of chunks stays within STM limits (~7±2); expertise means each chunk in LTM corresponds to *more* pieces. Superior memory thus reflects domain knowledge, not raw capacity.

## Where the pure-chunk account breaks

Later work overturned Chase & Simon's *STM-storage* claim:

- Even briefly presented (5 s) info is stored in **LTM, not STM** (Charness 1976; Frey & Adesman 1976).
- Experts show **residual** superiority even on *random* stimuli in several domains — inconsistent with a strict chunk-in-STM model.
- Experts build an **integrated, hierarchical representation** with relations to familiar game types, not just isolated chunks.

The modern successor is [[Long-Term Working Memory]] (via [[Skilled Memory Theory]]), which explains the same and broader phenomena with one mechanism.
