# Paradigm-Driven Research

A research strategy that accumulates knowledge by systematically varying a single established procedure to investigate a question or test a theory—rather than redesigning methodology wholesale for each study.

## Definition

Paradigm-driven research:
- **Reuses existing methods and materials** across multiple studies
- **Systematically alters specific procedural elements** by design (not accident)
- **Incorporates both replication and extension** in a single experimental program
- **Balances novelty and replication**: Extends knowledge while confirming core effects

Contrasts with:
- **Conceptual replication**: Changes operationalizations to test theoretical generality
- **Wholesale reinvention**: Creating new methods for each new question

## Mechanism

The strategy works by:

1. **Establishing a paradigm**: A validated experimental procedure that reliably demonstrates a phenomenon
2. **Systematic variation**: Deliberately modify one aspect (e.g., participant age, emotional state, temporal framing) to test new questions
3. **Replication component**: Each variation implicitly replicates the core finding under the modified condition
4. **Extension component**: The variation tests boundary conditions, moderators, mechanisms

## Classic Example: Deese-Roediger-McDermott (DRM) Paradigm

The DRM paradigm studies false memories: participants study lists of semantically related words and often falsely recall the non-presented critical lure.

**Base paradigm**: Roediger & McDermott (1995) established the DRM procedure and demonstrated robust false memory effect.

**Systematic variations** tested how:
- **Aging** influences false memory (Butler et al., 2004)
- **Mood** affects false memories (Storbeck & Clore, 2005)
- **Expectations** shape false memory frequency (Schacter et al., 1999)

Each variation:
- **Replicates** the original finding (false memories occur across conditions)
- **Extends** understanding (false memory increases with aging; decreases with sadness)
- **Identifies boundary conditions** and mechanisms

## Why Paradigm-Driven Research Addresses Scientific Inefficiency

### Problem It Solves

In typical practice:
- Each new study reinvents methodology based on published description
- Reinvention introduces unintended differences between studies
- Results in scattered literature where similar phenomena produce inconsistent findings
- Encourages researchers to continually innovate rather than build on prior work

### How Paradigm-Driven Approach Fixes This

1. **Replicability**: Reusing exact procedures ensures direct comparability across studies
2. **Efficiency**: Researchers don't waste effort recreating or learning from scratch
3. **Transparency**: Single methodology makes analysis flexibility more visible
4. **Cumulation**: Findings naturally build on each other within a coherent framework

## Relation to [[Replication]]

Paradigm-driven research is **not** the same as direct replication, but incorporates it:

- **Direct replication**: Exact reproduction of original procedures to verify effect
- **Paradigm-driven variation**: Intentional, designed modification of one procedural element
- **Conceptual replication**: Changed operationalization to test theoretical generality

Paradigm-driven research combines the strength of direct replication (confirmatory evidence for core effect) with the strength of conceptual replication (theoretical insight into generality).

## Risk and Mitigation

### Primary Risk

Research questions can drift toward being **about the method itself** rather than the theory the method tests. Using a single methodology for a long time risks reifying idiosyncratic features of that methodology as being the phenomenon.

**Example**: Studying false memories only with word lists might conflate properties of word-list memory with properties of false memory generally.

### Mitigation

Conceptual replication provides "substantial added value":
- Demonstrates the effect with different operationalizations
- Ensures findings are theoretically general, not methodologically idiosyncratic
- Proves the phenomenon isn't an artifact of the paradigm

**Strategy**: Combine paradigm-driven work within a methodology (depth) with periodic conceptual replications across methodologies (breadth).

## Practical Requirements

For paradigm-driven research to work:

1. **Paradigm availability**: Authors must make procedures, materials, and tools accessible to others
   - Shared repositories, video demonstrations, code
   - [[Open Methods and Tools|Open documentation of methodology]]

2. **Standards and conventions**: Clear specification of what varies and why
   - Deliberate alteration, not accident
   - Documented justification for modifications

3. **Research culture**: Value for building on others' methods
   - Reward for replication + extension
   - Prestige for establishing a productive paradigm

## Contrasts with Current Practice

| Aspect | Current | Paradigm-Driven |
|--------|---------|-----------------|
| **Novelty** | Every study should be "new" | Novelty through systematic extension of existing methods |
| **Methods** | Reinvented per study | Shared, reused, deliberately modified |
| **Replication** | Discouraged as "unoriginal" | Built-in via each variation |
| **Cumulation** | Scattered findings | Coherent program building knowledge |
| **Flexibility** | Many different approaches | Controlled variation within established framework |

## Relation to Open Science

Paradigm-driven research **requires and enables** [[Open Science]] practices:

- **Open methods**: Detailed sharing of procedures for others to use/adapt
- **Shared materials**: Making experimental tools available (questionnaires, stimuli, software)
- **Paradigm visibility**: Communities aware of "canonical" procedures (e.g., DRM paradigm)

In turn, paradigm-driven approach supports open science goals by:
- Making analysis transparency easier (one standardized method)
- Enabling replication (clear, shared baseline)
- Reducing waste from reinvention
