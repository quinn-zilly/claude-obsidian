# Christian Bizer

Christian Bizer is a computer scientist and one of the founders of the Linked Open Data movement. He co-developed the Linked Data publishing principles and is a key contributor to the DBpedia project.

## Affiliation

University of Mannheim (Data and Web Science Group), Mannheim, Germany.

## Key Contributions

- **[[lehmann-2015-dbpedia|DBpedia (2015, with Jens Lehmann et al.)]]** — core contributor and co-author
- **Linked Data publishing principles** — with Tim Berners-Lee; defined best practices for publishing and connecting data on the web
- **DBpedia Spotlight** — entity linking component that identifies Wikipedia entities in free text
- Research on Web data management, schema.org, and web table extraction

## Significance

Bizer's work bridging RDF/semantic web standards and practical web-scale applications made him a central figure in the LOD cloud. DBpedia's position as the LOD hub is partly a result of his focus on integration and interoperability.
