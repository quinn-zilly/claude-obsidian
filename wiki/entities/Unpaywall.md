# Unpaywall

Open database of OA status for scholarly articles, keyed by DOI. The de facto standard for determining whether a paper is open and by which route.

Source of the **five-category OA classification** (gold / green-only / hybrid / bronze / closed) used across large-scale OA studies — see [[OA Type Classification]]. Originates with Piwowar et al. (2018), "The state of OA."

In [[basson-et-al-2022-oa-dimensions-wos]]: provides OA status for both databases. [[Web of Science]] records are linked to Unpaywall; [[Dimensions]] ships Unpaywall status natively.
