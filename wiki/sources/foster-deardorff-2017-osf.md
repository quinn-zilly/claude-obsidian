# Foster & Deardorff (2017) — Open Science Framework (OSF)

> [!key-insight] Core Claim
> The OSF is a free, open, centralized research workflow tool that supports the entire research lifecycle — from hypothesis generation to publication — with project management, version control, preregistration, and integrations with existing scholarly tools.

## Bibliographic Details

Erin D. Foster and Ariel Deardorff. "Open Science Framework (OSF): Supporting Research from Hypothesis to Publication." *Journal of the Medical Library Association* 105, no. 2 (2017): 203–206.

This is a **resource review** article, not primary research. It describes and evaluates OSF as a library resource.

## Summary

### What Is the OSF?

The Open Science Framework (osf.io) is a free, open-source research workflow and collaboration platform developed by the [[Center for Open Science]] (COS), a non-profit organization. COS was founded by [[Brian Nosek]] with a mission to increase openness, integrity, and reproducibility of research.

### Core Features

| Feature | Description |
|---------|-------------|
| **Project management** | Hierarchical projects with components; wiki pages for documentation |
| **File storage** | Version-controlled file hosting; free up to 5 GB per private project |
| **Collaboration** | Shared workspaces with granular access controls (admin, read/write, read-only) |
| **Preregistration** | Timestamped, immutable registration of study designs and analysis plans |
| **Registrations** | Freeze any project state as a public, citable, permanent snapshot |

### Preregistration on OSF

OSF is the primary infrastructure for preregistration in psychology and social sciences:
- Researchers file a timestamped preregistration before data collection
- Multiple templates available (e.g., OSF Prereg Challenge template, clinical trial template, qualitative template)
- Public preregistrations are indexed and searchable
- Preregistration + Registration pairs allow comparison of planned vs. actual analysis

### Integrations

OSF connects to researchers' existing tools:
- Zotero, Mendeley (citation management)
- GitHub, GitLab, Bitbucket (code versioning)
- Google Drive, Dropbox, Box (file storage)
- Dataverse, figshare, S3 (data archiving)
- ORCID (researcher identification)

### The Psychology Reproducibility Project as Flagship

The paper cites [[open-science-collaboration-2015]] (OSC 2015) as the flagship demonstration of OSF: the 100-replication project was entirely coordinated through OSF, with all materials, data, and reports publicly archived, enabling post-hoc audit and re-analysis.

### Relevance for Librarians

Foster & Deardorff wrote for a library audience, emphasizing OSF as a tool for research data management (RDM) that librarians can recommend or support. They note OSF addresses funder mandates for data management plans (NIH, NSF, Wellcome Trust requirements described in [[munafo-2017-manifesto]]).
