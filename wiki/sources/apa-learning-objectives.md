# APA — Guidance for Writing Behavioral Learning Objectives

> [!key-insight] Core Claim
> Well-written behavioral learning objectives are observable, measurable, learner-focused, and linked to program content. Use action verbs. Avoid vague mental-state verbs ("know," "understand," "appreciate").

## Overview

One-page practitioner guidance document from the APA Education Directorate for continuing education (CE) program developers. Provides a checklist, verb taxonomy, and worked example.

## Checklist for Well-Written Learning Objectives

Learning objectives should be:
- **Observable and measurable** — use action verbs describing measurable behaviors
- **Outcome-focused** — describe what the learner will *know or be able to do* as result of attending
- **Learner-focused** — about the learner, not the instructor or content
- **Appropriate in breadth** — 3–4 objectives for a 4-hour program
- **Sufficient in depth** — postdoctoral level for psychologists; clearly articulated
- **Linked** to: (a) program narrative, (b) adequate supporting references, (c) promotional materials

## Verb Taxonomy

### ✓ Use These (Observable/Measurable)
| Level | Verbs |
|---|---|
| Knowledge/Recall | list, describe, recite, write, identify |
| Comprehension/Application | compute, discuss, explain, predict |
| Application | apply, demonstrate, prepare, use |
| Analysis | analyze, design, select, utilize |
| Synthesis | compile, create, plan, revise |
| Evaluation | assess, compare, rate, critique |

### ✗ Avoid These (Non-Observable)
- know, understand
- learn, appreciate
- become aware of, become familiar with

## Worked Example (4-hour CE on hypnosis and pain)

> At the end of this workshop, the learner will be able to:
> 1. Describe 3 key aspects of basic hypnosis theory and technique
> 2. Explain differences between demonstrations of hypnotic technique and phenomena
> 3. Identify 2 differences between acute and chronic pain
> 4. Demonstrate effective use of hypnosis in controlling acute pain

Note: Each objective uses a distinct action verb and specifies observable behavior.

## Connections to Learning Theory

This guidance operationalizes Bloom's taxonomy principles at the practitioner level. The verb categories roughly map to Bloom's levels (recall → comprehension → application → analysis → synthesis → evaluation). The emphasis on observable behaviors aligns with behaviorist and performance-based instructional design traditions.

**Link to transfer:** Explicitly stated learning objectives function as advance organizers (Kraiger et al. 1995 in [[burke-hutchins-2007-training-transfer]]) and signal to trainees what behaviors are expected post-training — a key predictor of transfer.

**Link to AERs:** Behavioral objectives provide explicit criteria against which after-event review can assess both success and failure (see [[ellis-davidi-2005-after-event-reviews]]).
